# 🥗 SafeBite AI

SafeBite AI is an AI-powered food hygiene and nutrition analysis web application that helps users make informed food choices.

## 🚀 Features

- 📸 Upload Food Image
- 🤖 AI Food Detection (MobileNetV2)
- 🔥 Calorie Analysis
- 🥩 Protein Analysis
- 🧪 Hygiene Score
- ❤️ Health Rating
- 💡 Food Recommendation
- 🎨 Modern User Interface

## 🛠️ Tech Stack

- Python
- Flask
- TensorFlow
- MobileNetV2
- HTML
- CSS

## 📂 Project Structure

```
SafeBite-AI
│
├── app.py
├── food_detector.py
├── requirements.txt
├── templates/
├── static/
└── README.md
```

## ⚙️ Installation

```bash
git clone https://github.com/Ninad534-cloud/SafeBite-AI.git

cd SafeBite-AI

pip install -r requirements.txt

python app.py
```
